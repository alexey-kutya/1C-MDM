Группы обмена справочной информацией

Справочник «Группы обмена справочной информацией» предназначен для организации группировки узлов обмена, используется в справочнике «Обмен справочной информации».

Справочник имеет линейную структуру. По команде "Создать" открывается дополнительная диалоговая форма для ввода нового элемента в справочник.

Реквизиты:

Код

Идентификационный код группы узлов обмена в системе

Наименование

Указывается наименование группы.
