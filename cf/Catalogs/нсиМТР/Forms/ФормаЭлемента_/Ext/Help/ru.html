Справочник "МТР"

Справочник материально-технических ресурсов является основным хранилищем информации об активах типа "Номенклатура", к которым относятся необходимые для обеспечения деятельности Предприятия материалы, оборудование, запчасти, комплектующие изделия и пр., приобретаемые со стороны или производимые своими силами. К ключевым атрибутам записей в справочнике МТР относятся наименование МТР, марка или другое условное обозначение, ссылка на нормативный или нормативно-технический документ, базовая единица измерения, производитель и различные признаки, характеризующие статус записи.

В качестве основного Классификатора продукции, представленной в справочнике МТР, используется классификатор, построенный на онтологической модели классификации. Такой классификатор не только обеспечивает структуризацию записей справочника, но и позволяет предложить пользователю расширенные возможности работы со справочниками, в частности, поддерживает поиск МТР по техническим характеристикам, поиск аналогов и взаимозаменяемого оборудования.

Единый справочник МТР необходим для решения задач тех бизнес-процессов предприятия, в которых используются данные по материально-техническим ресурсам. Справочник МТР может быть составной частью производственных и учетных систем, которые обеспечивают:

  • Управление финансами;
  • Управление закупками;
  • Управление затратами;
  • Управление продажами (сбытом);
  • Управление капитальным строительством;
  • Учет и отчетность.

Пользователями справочника МТР могут быть специалисты различных структурных подразделений предприятия, например:

  • Подразделений материально-технического снабжения;
  • Финансовых и бухгалтерских подразделений;
  • Складских подразделений;
  • Планово-экономических подразделений;
  • Производственных подразделений;
  • Ремонтных подразделений;
  • Проектных подразделений.

По справочнику ведется специализированное редактирование объектов НСИ "по заявкам", поэтому для позиций указываются: тип позиции, эталонная позиция (если тип позиции - дублирующая запись), признак использования, возможность нормализации и комментарий. Коме того, по справочнику ведется "дополнительная классификация", позволяющая определить несколько параллельных иерархических представлений или произвольных группировок.

Реквизиты:

Краткое наименование

Краткое наименование номенклатуры (Строка)

Полное наименование

Полное описание номенклатурной позиции достаточное для однозначной идентификация позиции всеми пользователями справочника МТР (Строка)

Единица измерения

Поле содержит базовое значение единицей измерения. При необходимости номенклатуре могут быть назначены альтернативные единицы измерения, для каждой из которых определяется коэффициент пересчета относительно базовой единицы. (Единицы измерения)

Класс

Название базового класса, к которому отнесена позиция (Классификатор продукции)

Каталожный номер

Поле обязательно для заполнения при наличии уникального каталожного номера производителя. (Строка)

Производитель

Текстовое наименование производителя данной номенклатуры (Строка). Поле обязательное для заполнения, если поле «Каталожный номер» заполнено.

Нормативно-технический документ

Поле обязательное для заполнения, в случае если производство данной номенклатуры регламентировано нормативным документом (ГОСТ, ОСТ, ТУ)

Так же в справочнике можно указать "Дополнительные реквизиты и сведения".
