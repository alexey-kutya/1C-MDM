Справочник "Классификатор продукции"

Классификация необходима, прежде всего, для структуризации и систематизации данных. В данной конфигурации применяется онтологический метод классификации. Наибольшие преимущества этот метод предоставляет при использовании справочника МТР. Пользователь получает расширенные возможности при работе со справочником:

  • разнообразные виды поиска материала, по свойствам и значениям;
  • подбор взаимозаменяемых материалов;
  • подбор материалов с одинаковыми характеристиками, но выпускаемых разными производителями.

 

Классификатор продукции построен на формальном выделении основных свойств и характеристик объектов МТР, и использовании их для построения классификационных структур и отношений между элементами этих структур.

Для каждой позиции справочника МТР, при использовании классификатора продукции, выделяется или создается класс по принципу однородности набора присущих данному объекту свойств, и формируется описание, включающие все свойства и их значения для данного материала. Онтологическая классификация обеспечивает непротиворечивое накопление любого количества информации в стандартной структуре классификации.

Такой метод классификации позволяет:

  • формализовать описания свойств каждого класса, обеспечить их непротиворечивость и требуемый уровень детализации;
  • достичь полноты и правильности наименований классифицируемых объектов (например, наименований материалов, сырья и готовой продукции в соответствующих справочниках), с учётом существенных особенностей и характеристик;
  • выделять описательные срезы из всего набора формализованных свойств, например, для материалов выделять "независимые описания", не содержащие указаний на конкретного производителя.

 

Основу классификатора продукции составляют базовые классы. Базовые классы содержат описательный шаблон для всех объектов, входящих в этот класс, включая наименование базового класса, набор всех возможных характеристик МТР, а для каждой характеристики МТР и множество её значений.

Каждый класс должен, по возможности, представлять собой группу однородных объектов. Каждый материал может принадлежать только одному классу и имеет определенный набор свойств и их значений.

Характеристики МТР и их значения описывают качественные и количественные свойства объектов, принадлежащих к данному классу.
