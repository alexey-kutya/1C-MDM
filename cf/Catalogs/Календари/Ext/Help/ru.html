Справочник "Календарные графики"

В справочнике "Календарные графики" хранится список календарных графиков, которые используются для определения сроков исполнения задач и фактической длительности рабочего времени, затраченного на выполнение задач. Графики строятся на основе производственных календарей. В справочнике содержится один предопределённый основной график, который применяется для всех по умолчанию, и индивидуальные графики пользователей.
