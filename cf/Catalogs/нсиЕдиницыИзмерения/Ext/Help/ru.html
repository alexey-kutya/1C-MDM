Справочник "Единицы измерения"

Справочник "Единицы измерения" предназначен для ведения списка всех единиц измерения, определенных для позиций МТР. Каждая позиция характеризуется базовой единицей измерения. При необходимости номенклатуре могут быть назначены альтернативные единицы измерения, для каждой из которых определяется коэффициент пересчета относительно базовой единицы.
