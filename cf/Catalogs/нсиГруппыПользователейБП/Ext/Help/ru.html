Справочник "Группы пользователей бизнес-процессов"

Справочник "Группы пользователей бизнес-процессов" предназначен для хранения групп пользователей, предназначенных для организации адресации задач. Пользователям системы группы назначаются в регистре сведений "Соответствие пользователей и групп пользователей БП". Если пользователю не назначена группа, то он по умолчанию относится к группе "Основная". При распределении задач "По роли и по группе пользователей БП" или "По группе пользователей БП" группа пользователя используется для поиска исполнителей в регистре сведений "Исполнители задач".
