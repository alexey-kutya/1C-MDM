Справочник "Домены"

Открыть справочник можно по соответствующей команде из раздела программы.

Справочник необходим для классификации отображаемых справочников и определяет их положение в "Панели справочников".

Для создания Домена необходимо нажать кнопку Создать, после чего откроется окно, в котором заполняется его Наименование.

Далее указывается цифра, определяющая порядок отображения справочников в Панели справочников.

Для добавления и выбора необходимых справочников в создаваемую группу нажимается кнопка Добавить.
