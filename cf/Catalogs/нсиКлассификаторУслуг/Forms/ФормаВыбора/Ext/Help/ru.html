Справочник "Банковские счета"

В справочнике "Банковские счета" хранятся сведения о банковских счетах юридических и физических лиц партнеров (контрагентов), с которыми работает торговое предприятие. Информация о расчетных счетах заполняется в справочнике "Контрагенты". Для каждого юридического и физического лица (контрагента) может быть введено любое количество расчетных счетов.

При вводе нового банковского счета получателя указывается информация, характеризующая банковский счет: номер счета, вид счета, валюта денежных средств, банк, в котором открыт счет и банк – корреспондент для непрямых расчетов, а также дополнительная информация, необходимая для корректного заполнения платежных документов по выбранному счету.  

По справочнику ведется специализированное редактирование "по заявкам", поэтому для позиций указываются: тип позиции, эталонная позиция (если тип позиции - дублирующая запись), признак использования, возможность нормализации и комментарий. Коме того, по справочнику ведется "дополнительная классификация", позволяющая определить несколько параллельных иерархических представлений или произвольных группировок.
