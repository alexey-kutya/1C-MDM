Алгоритмы разбора данных для загрузки

Справочник «Алгоритмы разбора данных для загрузки» предназначен для хранения типовых отрезков кода (алгоритмов), используемых в процессе считывания данных при загрузке из файла в систему Обработка «Загрузка данных».

Справочник имеет иерархическую структуру. Команда "Создать" позволяет добавить новый элемент справочника.

Реквизиты:

Родитель

Реквизит, предназначенный для группировки алгоритмов.

Наименование

Указывается наименование алгоритма.

Код

Типовой отрезок кода на встроенном языке платформы 1С для разбора данных.

Функциональность системы позволяет проверить работу введенного алгоритма в диалоговом окне.
