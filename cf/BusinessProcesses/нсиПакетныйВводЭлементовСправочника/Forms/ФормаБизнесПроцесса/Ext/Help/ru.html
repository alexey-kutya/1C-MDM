Пакетный ввод элементов справочника

По нажатию этой кнопки открывается диалоговое окно для формирования задания пакетного ввода элементов МТР.

Пользователь может инициировать заявки на добавления сразу нескольких позиций справочника. 

По команде Создать открывается диалоговое окно заявки, в рамка которой можно указать несколько новых позиций. Для каждой из них предусмотрена возможность заполнения Таблицы переходных ключей (см. п. 5.8.6. Настройки переходных ключей). По команде Отправить в службу НСИ заявка заноситься в систему.
