"Панель справочников"

Открыть панель справочников можно по соответствующей команде из раздела программы.

Панель справочников позволяет работать с справочниками из одного места и группирует их по группам. Работа с панелью справочников ведется в двух режимах:

  • в режиме просмотра;
  • в режиме настройки.

Режим просмотра

Панель справочников, состоит из двух частей. В нижней части перечисляются отобранные пользователем справочники, в верхней части отражается группа справочников
с функцией быстрого доступа.

  • Группы панели справочников соответствуют настроенным доменам.
  • Также в панели справоников могут быть предусмотрены и другие виды группировки, которые зависят от конкретного справочника.  
  • Нажмите Настройка или клавишу F2, для того чтобы перейти в режим настройки панели отчетов.

Режим настройки

  • Перейти в этот режим можно с помощью кнопки Настройки (клавиши F2).
  • С помощью нажатия на гиперссылку можно управлять видимостью справочника в группе.
  • Кнопками Переместить вверх/Переместить вниз можно изменить местоположение надписи.
  • При помощи контекстного меню по правой кнопке мыши справочник можно переместить в быстрый доступ:
    • Для настройки быстрого доступа щелкните по справочнику в списке правой кнопкой мыши, выберите команду Переместить в быстрый доступ;
    • Для отмены быстрого доступа к варианту отчета щелкните по нему правой кнопкой мыши, выберите команду Убрать из быстрого доступа.
  • Нажмите Настройки (клавишу F2), для того чтобы сохранить изменения и перевести панель отчетов в режим просмотра.
