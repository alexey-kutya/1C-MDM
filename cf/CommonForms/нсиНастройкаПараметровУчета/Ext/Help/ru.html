Форма отображает базовые настройки конфигурации и определяет базовое представление интерфейса и функционала по работе с системой.

К параметрам учета конфигурации относятся:

Заголовок системы

В данном поле указывается заголовок системы, который будет выводится в шапке окна программы.

Использовать дополнительные реквизиты и сведения

Данный реквизит отвечает за переключение режима использования дополнительных атрибутов справочников и классификаторов. Посредством установки этого флага можно включить механизмы поддержки ведения объектов НСИ с учетом дополнительных свойств и характеристик, добавляемых «на лету» атрибутов (реквизитов) справочников.

Использовать версионирование объектов

Данный поле переключает режимы работы с версиями объектов НСИ. Версионирование объектов НСИ позволяет хранить историю изменения значений атрибутов (реквизитов) объектов.

Использовать внешних пользователей

Данное поле отвечает за возможность доступа внешних пользователей к системе.

Тип хранения файлов

С помощью этого реквизита определяются варианты хранения файлов, прикрепляемых к заявкам на изменение/добавление информации и объектам НСИ.

Извлекать тексты файлов на сервере

Реквизит задает метод извлечения текстовых данных из прикрепленных файлов.

Максимальный размер файла

Значение установленное в данном поле определяет максимальный возможный размер прикрепляемого файла.

Запрет загрузки файлов по расширениям

С помощью этого поля задаются ограничения на типы прикрепляемых файлов.

Ограничивать доступ на уровне записей

Установка данного флага позволяет при настройке прав доступа пользователей вводить ограничения на доступ к конкретным записям.

Адресаты получения отчета по журналу регистрации

В данном реквизите указываются пользователи (обычно- администраторы системы), ответственные за контроль ведения журнала регистрации.

Путь к внешней компоненте

В поле указывается путь к внешней компоненте.

Вести ручную обработку справочников

Реквизит регулирует режим работы с объектами НСИ при добавлении/изменении данных: ручной или по заявкам.

Учитывать в разрезе дополнительных классификаторов

Реквизит отвечает за переключение возможности использования дополнительных классификаторов (иерархических группировок) при классификации объектов НСИ.

Использовать блок «МТР»

Установка данного флага позволяет включить возможность использования блока объектов НСИ, описывающих Материально-Технические Ресурсы (МТР).

Учитывать в разрезе классификатора продукции

Данный флаг отвечает за переключение возможности использования классификатора продукции и связанных с ним свойств и характеристик при классификации объектов НСИ, описывающих Материально- Технические Ресурсы.

Использовать блок «Услуги»

Установка данного флага позволяетвключить возможность использования блока объектов НСИ, описывающих услуги.

Учитывать в разрезе классификатора услуг

Данный флаг отвечает за переключение возможности использования классификатора услуг и связанных с ним свойств и характеристик при классификации объектов НСИ, описывающих услуги.

Использовать блок «Контрагенты»

Установка данного флага позволяетвключает возможность использования блока объектов НСИ, описывающих контрагентов.

Учитывать в разрезе договоров

Данный флаг отвечает за включение/выключение механизмов использования и ведения справочника «Договоры» и связанных с ним вспомогательных справочников.

Учитывать в разрезе банковских счетов

Данный флаг отвечает за включение/выключение механизмов использования и ведения справочника «Расчетные счета» и связанных с ним вспомогательных справочников.

Использовать бизнес- процессы и задачи

Данный флаг отвечает за использование механизмов подсистемы Управление заявками на добавление/ изменение объектов НСИ в процессе ведения.

Использовать дату начала задач

С помощью данного флага включается возможность использования даты начала задач в подсистеме Управления заявками на добавление/ изменение объектов НСИ в процессе ведения.

Изменять задания задним числом

Реквизит регулирует возможность изменения задним числом заданий (запросов) при добавлении/изменении объектов НСИ в процессе ведения.

Дата уведомления о новых задачах

С помощью этого реквизита устанавливается дата и время уведомления о новых задачах.

Использовать дату и время в сроках задач

Реквизит регулирует использование даты и времени в задачах.

Старший эксперт

В данном поле указывается пользователь системы: Руководитель службы НСИ.

Использовать полнотекстовый поиск

Реквизит отвечает за переключение режимов использования интегрированного в платформу механизма полнотекстового поиска.

Метод сравнения строковых данных

Реквизит устанавливает используемый при нечетком поиске метод сравнения искомого образца и строковых атрибутов справочников.

Количество фоновых заданий при поиске

Реквизит устанавливает максимальное количество фоновых заданий при поиске данных в клиент-серверном режиме работы.

Размер N-грамм

Устанавливает размер N-грамм для алгоритмов нечеткого поиска.

Использовать обмен данными

С помощью этого реквизита регулируется включение интеграционных возможностей системы.

Префикс узла распределенной информационной базы

В данном реквизите задается префикс информационной базы в распределенной конфигурации ЕС НСИ.
