Регистр сведений "Соответствие пользователей и групп пользователей БП"

В регистре настраивается принадлежность пользователей к группам пользователей бизнес-процессов. Один пользователь может входить в различные группы. Флаг "Использовать по умолчанию" определяет, какая из групп указывается по умолчанию. Если для пользователя не указана группа, то по умолчанию считается, что он входит в предопределённую группу "Основная".
