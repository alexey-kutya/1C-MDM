Доступность операций по типам объектов НСИ

В данном списке содержится информация о возможности использования ручного ввода элементов в конкретных объектах системы НСИ.

При администрировании системы пользователь должен указать, существует ли возможность ввода информации в конкретный объединенный справочник в базе 1С:MDM непосредственно функционалом данного приложения или эта информация подлежит только трансляции из информационных баз – источников в информационные базы – потребители.

Реквизиты:

Тип объекта

Поле выбора из списка справочников, включенных в базу 1С:MDM. В данном поле указывается программное наименование объекта Справочник, для которого определяется доступность ручного ввода элемента средствами 1С:MDM.

Использовать ручную обработку справочников

Флаг, установка которого открывает доступ к возможности ручного ввода элемента в данном справочнике средствами 1С:MDM.
