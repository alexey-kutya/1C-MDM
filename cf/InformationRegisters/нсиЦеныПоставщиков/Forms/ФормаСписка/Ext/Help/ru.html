Цены поставщиков

В данном справочнике содержится информация о ценах на данную номенклатуру у разных поставщиков. При расценке материально-технических ресурсов пользователь может указать не только текущую стоимость МТР, но и для какой организации будет поставляться МТР по указанной цене. В представленном функционале конфигурации предусмотрено указание периода, в течение которого будет действительна указанная цена.

Реквизиты:

МТР

Наименование объекта МТР, для которого заполняется цена. По умолчанию данное поле заполняется автоматически кратким наименованием объекта, из которого вызван Справочник «Цены поставщиков»

Период

Период, в течение которого будет возможно закупать данную номенклатуру по указанной цене.

Организация

Поле выбора из списка организаций. В данном реквизите указывается организация, для которой осуществляется поставка МТР данным поставщиком по указанной цене.

Поставщик

Поле выбора из списка поставщиков из справочника Контрагентов

Валюта

В данном атрибуте указывается валюта стоимости МТР у данного поставщика

Цена

Числовое поле, в котором указывается размер стоимости МТР
