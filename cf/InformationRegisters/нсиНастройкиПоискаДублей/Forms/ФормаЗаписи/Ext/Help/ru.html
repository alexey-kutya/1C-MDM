Регистр сведений "Настройки поиска дублей"

Данный регистр предназначен для хранения настроек отборов, которые применяются для поиска дублей в функциональных справочниках при выполнении заявок на ввод нового элемента справочника. Если в настройке отбора не заполнено правое значение, то при поиске дубля оно будет автоматически подставляться из соответствующего реквизита макета элемента справочника.
