Регистр сведений "Назначение бизнес-процессов справочникам"

Данный регистр позволяет для каждого функционального справочника назначить настройки бизнес-процессов, которые будут использоваться для изменения содержимого справочника.

Измерения:

ИмяСправочника - имя справочника, для которого назначается настройка;

НастройкаБП - назначенная настройка;

Заявитель - ссылка на пользователя или группу пользователей, для которых будет выбираться указанная настройка.

Ресурсы:

Использовать - флаг, указывающий, что настройка будет применяться;

ОтборПредметов - ссылка на настройки поиска данных, которые определяют возможность использования настройки для конкретной позиции функционального справочника.
