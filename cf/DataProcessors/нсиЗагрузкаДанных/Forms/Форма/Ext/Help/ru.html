Обработка "Загрузка данных"

Универсальная обработка "Загрузка данных" предназначена для загрузки данных в систему без заявок во все справочники системы. Обработка предоставляет сервисы по загрузки информации произвольного формата из excel файлов, а так же позволяет подгружать данные в систему из произвольной базы данных.

Функционал обработки позволяет вводить групповые заявки в систему из excel файлов.

Пользователю предоставлена возможность сохранять и восстанавливать параметры и настройки загрузки данных из разных шаблонов загрузки.

Работа по загрузки данных в систему из файла разбита на 7 шагов.

  • На первом шаге пользователь должен определить параметры подключения к файлу, из которого предполагается загрузка информации.
  • На втором шаге пользователь определяет список файлов, содержащих необходимую информацию. В случае загрузки данных из базы данных данный шаг пропускается.
  • На третьем шаге необходимо определить целевую таблицу загрузки и указать параметры действий с загружаемыми данными. В случае, если необходимо при загрузки информации добавить только новые элементы в справочник или обновить ранее загруженные данные необходимо выбрать соответствующее значение формы работы с данными. По умолчанию значение формы работы с данными установлено на «Изменение или создание» новых элементов справочника. Универсальная обработка позволяет не только актуализировать данные основной (целевой таблицы) загрузки, но и обновлять данные в ссылочных таблицах и справочниках.
  • На четвертом шаге пользователю необходимо указать перечень полей, по которым в системе необходимо осуществлять поиск записей в справочнике. Здесь необходимо не только указывать перечень поисковых атрибутов для основного справочника, но и указать перечень поисковых реквизитов дополнительных ссылочных данный.
  • И только на пятом шаге пользователь определяет перечень реквизитов справочника, которые будут актуализироваться, с указанием номера колонки данного атрибута в фале загрузки. На данном этапе также можно выбрать алгоритм разбора данных. Данная опция становиться очень полезной при заполнении реквизитов конечного перечисления значений.
  • На шестом шаге пользователь может настроить заполнение реквизитов константными значениями.
  • Заключительный седьмой шаг определяет функции загрузки: загружать данные напрямую в справочник или создать групповую заявку на добавление данных, а также задать параметры вывода лога загрузки.
