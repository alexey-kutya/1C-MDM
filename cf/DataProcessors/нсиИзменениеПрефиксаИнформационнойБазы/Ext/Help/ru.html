Обработка "Изменение префикса информациооной базы"

Обработка предназначена для изменения префикса во всех справочниках и документах для продолжения нумерации.

  • Во всех данных, введенных в этой программе (со старым префиксом).
  • Во всех данных (даже если у них другой префикс).
  • Только в последних введенных актуальных элементах каждого списка.
  • Создать по одному пустому элементу с новым префиксом во всех списках.
